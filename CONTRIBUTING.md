# Contributing to JobPulse

Thanks for helping improve JobPulse.

## Development Setup

```bash
npm install
npm run dev
```

Type check before pushing:

```bash
npm run check
```

## Pull Request Guidelines

- Keep PRs focused and small
- Include a clear title and summary
- Add screenshots for UI changes
- Note any migration or behavior changes
- Confirm npm run check passes

## Commit Style

Conventional commits are recommended:

- feat: new user-facing feature
- fix: bug fix
- docs: docs only
- refactor: non-behavioral code changes
- chore: maintenance

## Areas Where Help Is Welcome

- Better ranking quality and explainability
- Job source resiliency
- Desktop packaging polish
- Accessibility and performance
- Tests and observability
